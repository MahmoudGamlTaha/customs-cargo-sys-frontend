# 🎨 دليل تغيير ألوان التطبيق

## 📋 نظرة عامة
تم تحديث التطبيق لاستخدام متغيرات CSS لتسهيل تغيير الألوان. الآن يمكنك تغيير اللون الأساسي للتطبيق من مكان واحد فقط!

## 🔧 كيفية تغيير الألوان

### الطريقة السهلة (الموصى بها):
1. افتح ملف `src/styles/colors.css`
2. غيّر قيمة `--brand-primary` في السطر 7:

```css
:root {
  --brand-primary: #EA580C;  /* غيّر هذا اللون */
}
```

### الألوان البديلة المتاحة:
- **برتقالي داكن**: `#EA580C` (الحالي)
- **أخضر داكن**: `#2D5016`
- **بنفسجي داكن**: `#6B46C1`
- **أحمر داكن**: `#B91C1C`
- **رمادي داكن**: `#374151`
- **أزرق داكن مختلف**: `#1E40AF`

### مثال على التغيير:
```css
/* لتغيير اللون إلى بنفسجي */
--brand-primary: #6B46C1;

/* أو استخدام متغير جاهز */
--brand-primary: var(--alt-primary-purple);
```

2. **ألوان بديلة جاهزة**: 5 ألوان مختلفة متاحة

## 🚀 كيفية التطبيق

1. غيّر اللون في `src/styles/colors.css`
2. احفظ الملف
3. أعد تحميل التطبيق
4. ستلاحظ التغيير في جميع العناصر التي تستخدم `brand-primary`

## 📝 ملاحظات مهمة

- التغيير يؤثر على جميع العناصر التي تستخدم `text-brand-primary` أو `bg-brand-primary`
- الرسوم البيانية ستتحدث تلقائياً
- لا حاجة لتعديل ملفات أخرى
- يمكن إضافة ألوان جديدة بسهولة في ملف `colors.css`
